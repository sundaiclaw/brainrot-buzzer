# Proposal: Brainrot Buzzer

## Why
Typing tests are usually sterile. The requested design is a joke game: brainrot phrases, speed pressure, and a loud punishment sound on every typo.

## What changes
Build a browser typing-speed game where AI generates absurd brainrot phrases, tracks WPM/accuracy, and fires a loud buzzer whenever the player types the wrong next character.

## Success criteria
- Player can start a timed run instantly.
- AI generates fresh phrase packs.
- Wrong key emits a loud noise immediately.
- Game shows WPM, accuracy, streak, and final score.
